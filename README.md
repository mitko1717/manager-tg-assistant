# Eurotrips Telegram Bot — Інструкція

AI-помічник для менеджерів Євротріпс. Ти надсилаєш питання клієнта у Telegram, бот генерує готову відповідь у стилі компанії.

---

## Як це працює

1. Ти отримуєш питання від клієнта
2. Копіюєш питання у спеціальний формат
3. Надсилаєш боту у Telegram
4. Бот повертає готову відповідь — копіюєш і відправляєш клієнту

---

## Як надсилати повідомлення боту

Формат повідомлення завжди такий:

```
НАЗВА_ТУРУ: Назва туру; ІМʼЯ: Імʼя клієнта; Питання клієнта
```

**Приклад з одним питанням:**
```
НАЗВА_ТУРУ: Вікенд у Будапешті + Відень; ІМʼЯ: Олена; Коли виїзд і скільки коштує тур?
```

**Приклад зі списком питань:**
```
НАЗВА_ТУРУ: Вікенд у Будапешті + Відень; ІМʼЯ: Олена;
1. Скільки людей у номері?
2. Чи є розетки в автобусі?
3. Яка страховка входить?
```

> ❗️ Важливо: крапка з комою `;` після назви туру і після імені — обовʼязкова

## Запуск бота

### Перший раз (встановлення)

Відкрий термінал у папці проєкту і виконай:

```bash
npm install
```

Створи файл `.env` у корені папки і встав туди:

```
TELEGRAM_BOT_TOKEN=твій_токен
OPENAI_API_KEY=твій_ключ
```

### Щоразу для запуску

```bash
npx ts-node bot.ts
```

Бот працює поки відкритий термінал. Щоб зупинити — натисни `Ctrl + C`.

### Запуск у продакшн режимі (стабільніше)

```bash
npx tsc
node dist/bot.js
```

```
│   ├── examples.md               # приклади діалогів
│   ├── eurotrips.txt             # загальна інформація про компанію
│   ├── faq.md                    # часті питання і відповіді

│   └── tours/
│       ├── vikend-v-budapeshti-viden.txt
│       └── ...                   # файли інших турів
```

## Як додати новий тур

1. Створи файл у папці `context/tours/`
2. Назва файлу — латиницею, малими літерами, пробіли замінити на `-`

| Назва туру | Назва файлу |
|---|---|
| Вікенд у Будапешті + Відень | `vikend-u-budapeshti-viden.txt` |
| Париж + Нормандія | `parizh-normandiia.txt` |
| Барселона – Мадрид | `barselona-madryd.txt` |

3. Заповни файл за шаблоном:

```
## Основні дані
- Дати: ...
- Вартість: ...
- Виїзд: ...

## Що входить у вартість
...

## Проживання
...

## Автобус та переїзди
...

## Кордон
...

## Група
...
```

> Якщо файл туру не знайдено — бот відповідає без інформації про конкретний тур, але загальні питання обробляє нормально.

## Контекстні файли — що і навіщо

| Файл | Що містить | Коли редагувати |
|---|---|---|
| `tone.md` | тон, стиль, заборонені слова | якщо хочеш змінити манеру спілкування |
| `rules.md` | правила формату відповідей | якщо хочеш змінити структуру відповідей |
| `examples.md` | приклади реальних діалогів | якщо хочеш показати боту нові приклади |
| `eurotrips.txt` | коротко про компанію | рідко |
| `faq.md` | відповіді на типові питання | якщо зʼявились нові часті питання |
| `dont.md` | що бот не повинен писати | якщо бот щось пише неправильно |

Усі файли необовʼязкові — якщо файл відсутній, його секція просто не потрапляє у відповідь.

---

## Якщо щось пішло не так

**Бот не відповідає на всі питання зі списку** — перевір що питання пронумеровані (`1.`, `2.`, `3.` і т.д.)

**Бот не знає деталей туру** — перевір що файл туру існує в `context/tours/` і назва файлу правильна

**Помилка при запуску** — перевір що файл `.env` існує і в ньому правильні токени

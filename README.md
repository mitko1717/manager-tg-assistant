# Eurotrips Telegram Bot

AI-бот для менеджерів Євротріпс. Отримує питання клієнта, завантажує контекст туру і генерує відповідь у стилі компанії.

---

## Встановлення

```bash
npm install
```

Створи `.env` файл:

```env
TELEGRAM_BOT_TOKEN=your_token
OPENAI_API_KEY=your_key
```

---

## Запуск

Розробка:
```bash
npx ts-node bot.ts
```

Продакшн:
```bash
npx tsc
node dist/bot.js
```

---

## Формат повідомлення у Telegram

```
НАЗВА_ТУРУ: Вікенд у Будапешті + Відень; ІМʼЯ: Олена; Питання клієнта
```

- `НАЗВА_ТУРУ` — використовується для пошуку файлу туру
- `ІМʼЯ` — імʼя клієнта, зʼявляється у привітанні
- Далі — довільний текст: одне питання, список, або кілька абзаців

---

## Файли турів

Назва файлу формується автоматично з назви туру — нижній регістр, пробіли та дефіси замінюються на `-`.

| Назва туру | Файл |
|---|---|
| Вікенд у Будапешті + Відень | `vikend-u-budapeshti-viden.txt` |
| Париж + Нормандія | `parizh-normandiia.txt` |
| Барселона – Мадрид | `barselona-madryd.txt` |

Якщо файл туру не знайдено — бот відповідає без інформації про тур, загальний контекст працює далі.

Щоб перевірити яку назву файлу шукає бот — додай в `bot.ts`:
```ts
console.log('Шукаю файл туру:', tourFile)
```

---

## Формат файлу туру

Рекомендована структура `context/tours/назва-туру.txt`:

```
## Основні дані
- Дати: ...
- Вартість: ...
- Виїзд: ...

## Що входить у вартість
...

## Проживання
...

## Автобус та переїзди
...

## Кордон
...

## Група
...
```

## Контекстні файли

| Файл | Що містить |
|---|---|
| `tone.md` | тон, стиль, заборонені слова |
| `examples.md` | приклади реальних діалогів менеджера |
| `eurotrips.txt` | коротко про компанію |
| `faq.md` | відповіді на типові питання |
| `dont.md` | що модель не повинна робити |

Усі файли необовʼязкові — якщо файл відсутній, його секція просто не потрапляє в промпт.
